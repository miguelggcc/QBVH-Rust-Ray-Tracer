Monte Carlo Ray Tracer written in Rust from scratch. It features:
* Multithreading
* Extremely fast Quad-BVH (Bounding Volume Hierarchy) with SIMD
* Vast material selection
* Triangle Rendering
* HDRI background
* Light Sampling
* Bloom effect
